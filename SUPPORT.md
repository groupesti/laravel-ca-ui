# Support

## Getting Help

- **Bug Reports**: [GitHub Issues](../../issues) — for confirmed bugs only.
- **Questions & Discussion**: [GitHub Discussions](../../discussions) — for usage questions, ideas, and general discussion.
- **Security Issues**: See [SECURITY.md](SECURITY.md) — do **not** open a public issue.

## Before Reporting a Bug

Please include:

- Package version
- PHP version (must be 8.4+)
- Laravel version (12.x or 13.x)
- Minimal reproduction steps or code snippet

## Note

GitHub Issues are **not** a general support forum. Use Discussions for questions.
